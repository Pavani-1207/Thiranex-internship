# Sales & Revenue Analysis Dashboard

## Internship
Thiranex Data Analytics Internship – Task 1

---

## Project Overview

This project focuses on analyzing sales and revenue data using Python.

The objective of this project is to:
- analyze sales performance
- generate KPI insights
- visualize revenue trends
- identify top-performing products and regions
- automate reporting and dashboard generation

The project helps businesses make data-driven decisions using sales analytics.

---

## Technologies Used

- Python
- Pandas
- Matplotlib
- OpenPyXL

---

## Dataset Used

Superstore Sales Dataset from Kaggle

Dataset includes:
- Sales records
- Product categories
- Customer details
- Regional sales
- Profit analysis
- Order information

---

## Project Features

### Data Cleaning
- Duplicate removal
- Missing value handling
- Date conversion
- Column standardization

### KPI Analysis
- Total Sales
- Total Profit
- Total Orders
- Top Product Category
- Top Sales Region

### Dashboard Visualizations
- Sales by Region
- Monthly Sales Trend
- Top 10 Products by Sales

### Automated Reporting
- KPI summary export
- Cleaned dataset generation
- Automated chart creation
- Text-based reporting

---

## Dashboard Workflow

1. Load sales dataset
2. Clean and preprocess data
3. Generate KPI metrics
4. Create visual analytics
5. Export reports and summaries
6. Generate automated dashboard outputs

---

## Visualizations Generated

### 1. Sales by Region
Analyzes sales performance across regions.

### 2. Monthly Sales Trend
Displays monthly revenue growth patterns.

### 3. Top 10 Products by Sales
Identifies highest-selling products.
