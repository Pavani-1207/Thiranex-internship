## Project Overview

This project automates the process of data cleaning, preprocessing, KPI analysis, and reporting using Python.

The system performs:
- Data cleaning
- Duplicate removal
- Missing value handling
- KPI generation
- Automated report creation
- Data visualization

This project helps improve efficiency in handling large datasets and generating business insights automatically.

---

## Technologies Used

- Python
- Pandas
- Matplotlib
- OpenPyXL

---

## Dataset Used

Superstore Sales Dataset from Kaggle

Dataset includes:
- Sales data
- Customer information
- Product categories
- Regional performance
- Profit analysis

---

## Features

### Data Cleaning
- Removes duplicate records
- Handles missing values
- Standardizes column names
- Converts date columns

### KPI Analysis
- Total Sales
- Total Profit
- Total Orders
- Top Performing Category
- Top Sales Region

### Automated Reporting
- Generates cleaned CSV file
- Creates KPI summary report
- Exports Excel summary
- Generates visualization charts automatically

---

## Visualizations Generated

### 1. Sales by Category
Shows sales performance across different product categories.

### 2. Monthly Sales Trend
Displays monthly sales growth and trends.

### 3. Profit by Region
Analyzes regional profitability performance.
