# Customer Segmentation Project

## Internship
Thiranex Data Analytics Internship – Task 2

---

## Project Overview

This project focuses on customer segmentation using Machine Learning techniques.

The objective of this project is to:
- analyze customer behavior
- segment customers based on spending patterns
- identify different customer groups
- generate business insights using clustering algorithms

Customer segmentation helps businesses improve marketing strategies and customer targeting.

---

## Technologies Used

- Python
- Pandas
- Matplotlib
- Scikit-learn
- OpenPyXL

---

## Dataset Used

Mall Customer Segmentation Dataset from Kaggle

Dataset includes:
- Customer ID
- Gender
- Age
- Annual Income
- Spending Score

---

## Project Features

### Data Cleaning
- Duplicate removal
- Missing value handling
- Data preprocessing

### Customer Segmentation
- K-Means Clustering
- Feature scaling
- Customer group identification

### Business Insights
- High spending customers
- Low spending customers
- Income-based segmentation
- Customer behavior analysis

### Automated Reporting
- Segmented customer CSV export
- Cluster summary generation
- Visualization charts
- Automated text report

---

## Machine Learning Workflow

1. Load customer dataset
2. Clean and preprocess data
3. Select important features
4. Apply feature scaling
5. Train K-Means clustering model
6. Create customer segments
7. Visualize clusters
8. Generate reports

---

## Clustering Features Used

- Annual Income
- Spending Score

These features help identify customer purchasing behavior and segmentation patterns.

---

## Visualization Generated

### Customer Segmentation Graph
Displays customer clusters based on:
- Annual Income
- Spending Score

Each cluster represents a different customer segment.
