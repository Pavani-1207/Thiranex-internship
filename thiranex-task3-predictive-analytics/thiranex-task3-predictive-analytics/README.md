# Predictive Analytics Using Historical Data

## Internship
Thiranex Data Analytics Internship – Task 3

---

## Project Overview

This project focuses on predictive analytics using historical Walmart sales data.

The objective of the project is to:
- preprocess historical data
- build a predictive model
- forecast future sales
- evaluate prediction accuracy
- generate automated reports and visualizations

The project uses Machine Learning techniques with Python and Scikit-learn.

---

## Technologies Used

- Python
- Pandas
- Matplotlib
- Scikit-learn
- OpenPyXL

---

## Dataset Used

Walmart Sales Forecast Dataset from Kaggle

Dataset contains:
- Weekly sales data
- Temperature
- Fuel prices
- Consumer Price Index (CPI)
- Unemployment rates
- Holiday indicators

---

## Project Features

### Data Preprocessing
- Missing value handling
- Duplicate removal
- Date conversion
- Data cleaning

### Predictive Modeling
- Linear Regression Model
- Historical sales forecasting
- Future trend prediction

### Model Evaluation
- Mean Absolute Error (MAE)
- R2 Score Analysis

### Automated Reporting
- Prediction CSV export
- Excel model summary
- Automated text report
- Forecast visualization

---

## Machine Learning Workflow

1. Load historical dataset
2. Clean and preprocess data
3. Select important features
4. Split training and testing data
5. Train Linear Regression model
6. Predict future sales
7. Evaluate model performance
8. Generate reports and charts
